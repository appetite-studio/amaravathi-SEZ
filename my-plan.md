# AIM States x Amaravati Startup Capital

## Prototype Web App Brief

We are building a cinematic prototype web application for AIM States.

This is NOT a production SaaS app.
This is a visual concept demo designed for investors.

The goal is to demonstrate what a future digital founder district / startup SEZ operating system could feel like.

The prototype is inspired by:

* Próspera
* Network states
* Digital residency systems
* Founder cities
* Modern government dashboards
* Apple-like clean interfaces
* Stripe / Linear / Notion quality UI

The app represents:
“Amaravati Startup Capital”
A founder-first startup district inside Amaravati, Andhra Pradesh.

Core concept:
A founder from anywhere in the world can:

* apply for residency
* register a startup
* join the community
* access founder infrastructure
* receive approvals
* interact with district systems
* attend events
* access benefits
  through one digital platform.

The platform should feel like:
“A digital operating system for a startup city.”

DO NOT build this like a normal dashboard template.
It should feel premium, futuristic, calm, minimal, government-grade but startup-native.

Important:
This is only a frontend prototype.
NO backend.
NO database.
NO authentication server.

Everything should use:

* localStorage
* mock JSON data
* fake realistic demo data
* static APIs
* simulated loading states

The prototype will be used inside a recorded investor video walkthrough.

The entire product should feel believable.
